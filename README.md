# 📈 Option Pricing and Risk Analysis using Data Science

## 📌 Overview

This project implements financial models to price options and analyze risk using real market data.

## 🚀 Features

* Option pricing using Black-Scholes model
* Risk analysis and backtesting
* Data visualization of market trends
* Python-based analytical tool

## 🛠 Tech Stack

* Python
* NumPy, Pandas
* Matplotlib
* SciPy

## 📊 Models Used

* Black-Scholes Option Pricing Model
* Risk Metrics Calculation

## 📈 Results

* Accurate pricing of options using real data
* Risk analysis through backtesting

## ▶️ How to Run

pip install -r requirements.txt
python option_pricing_app.py
